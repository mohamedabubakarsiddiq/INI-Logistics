# INI Logistics Portal - Test Plan

## 1. Introduction

### Project Name

INI Logistics Portal

### Objective

To verify that the INI Logistics Portal functions correctly and meets business requirements for shipment creation, shipment tracking, dashboard operations, profile management, and user authentication.

---

## 2. Scope

### In Scope

* Login Functionality
* Dashboard
* Shipment Creation
* Shipment Tracking
* Profile Management
* Logout Functionality

### Out of Scope

* Payment Gateway
* Third-Party Integrations
* Production Database Testing

The following testing types will be performed:

* Functional Testing
* UI Testing
* Smoke Testing
* Regression Testing
* Usability Testing
* Compatibility Testing
